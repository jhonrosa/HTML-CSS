Principais Formatações

Negrito / Itálico

Nesta frase temos um termo em negritousando a tag <b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase, temos a tag I para destacar o termo Italico (não semantico)

Nesta frase, temos um termo em ênfase usando a tag EM (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto Grande e Pequeno

Podemos criar um texto grande usando a tag <big> que é não semântica.

E também podemos usar a tag <small> para textos pequenos. (ainda usada para textos não tão importantes de rodapé).

Texto Deletado (riscado ao meio) e Sublinhado

Podemos usar a tag <del> para deixar o texto riscado ao meio.

Já com a tag <ins> conseguimos sublinhar o texto.

Texto Sobrescrito e Subscrito

Com a tag <sup> podemos elevar o texto que queremos. Ex: x10+y2.

Já com a tag <sub> podemos abaixar a letra ou texto que quisermos. Ex: H2O.
